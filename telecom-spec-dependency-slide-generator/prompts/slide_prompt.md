# Slide Generation Prompt

Generate an executive and technical slide deck that presents the anatomy of the detected dependencies.

## Required structure
1. Cover
2. Executive summary
3. Global dependency map
4. Domain clusters
5. Dependency matrix
6. Critical paths
7. Circular dependencies
8. Gaps and conflicts
9. One slide per document
10. Recommendations

## Visual guidelines
- use a clean corporate style;
- keep bullets short;
- use colors by domain;
- highlight risks in red or amber;
- avoid excessive text;
- prefer diagrams whenever possible.
